Anatomy of a Blog - update

by [Dave Kensinger]

2 May 2024

Here’s how I make this site:

  1. I write and edit posts in one text file
  2. I use Hugo to build the site
  3. I use github.io to host the site

That’s it!

Here’s a bit more detail of the steps above:

  1. I use Doom Emacs as my text editor. I write in one .org text file.
  2. I use an application called Hugo to build the site. I use a package within Emacs called ox-hugo that automatically builds and refreshes the site when I make any changes. I use (and sometimes change) the theme of the site using a hugo-based theme. Currently I use one called Typographyxs.
  3. Finally, I update the site on github. I use Git and Magit within Emacs to commit and push changes to the site.

The common thread above is Emacs. Aside from using Emacs for email, news, note taking, calendar, TODO lists, calculator, version control, time tracking and more… this blog is just another thing I can do within Emacs. It’s great.
